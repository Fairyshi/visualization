## 环境
### 浏览器支持
Three.js 可以在任何流行的浏览器中运行，除了IE大部分版本外。这些IE可以通过安装 Google Chrome Frame 或者 iewebgl 来支持 WebGL。

### 新渲染器
Threejs 正在开发遵从 CSS-3D规范的新渲染器，该规范被包括 IE10、手机、平板电脑、桌面在内的大多数浏览器支持。

### 开发服务器
```
// 1.安装全局 node server
npm install -g http-server

// 2.终端中带 源码的目录中启动server
http-server

// 3.浏览器中访问服务
http://127.0.0.1:8080
```
